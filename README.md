# Строительная компания ОДИ — одностраничный сайт на React + Vite

Одностраничный лендинг для строительной компании «ОДИ». 

## Технологии
- React 18
- Vite 6
- CSS (адаптивная сетка, плавная прокрутка)

## Структура
- `src/components` — React-компоненты (Header, Hero, ProjectsSection, PlaceholderSection)
- `src/data` — данные для слайдеров и карточек проектов
- `src/styles` — стили приложения

## Запуск
1. Установите зависимости:
   ```bash
   npm install
   ```
2. Запустите dev-сервер:
   ```bash
   npm run dev
   ```
3. Сборка проекта:
   ```bash
   npm run build
   ```
4. Просмотр production-сборки:
   ```bash
   npm run preview
   ```

## Примечания
В проекте использованы заглушки (цветные блоки) вместо реальных изображений. Их можно заменить на фотографии домов и визуальные материалы компании.
